# HTML 字符实体

在 HTML 中，某些字符是预留的。例如，不能直接使用小于号（`<`）和大于号（`>`），因为浏览器会误认为它们是标签。若要正确显示预留字符，必须使用字符实体（Character Entities）。

## 语法格式

```html
<!-- 使用实体名称 -->
&entity_name;

<!-- 使用实体编号 -->
&#entity_number;
```

:::tip 提示
使用实体名称便于记忆，但浏览器可能不支持所有实体名称。实体数字兼容性更好。
:::

:::warning 注意
实体名称对大小写敏感！
:::

## 实战应用场景

### 1. 在网页中显示 HTML 标签

```html
<!-- 错误示例：浏览器会渲染为实际标签 -->
<p>使用 <p> 标签创建段落</p>

<!-- 正确示例：显示为普通文本 -->
<p>使用 &lt;p&gt; 标签创建段落</p>
```

### 2. 显示代码中的特殊字符

```html
<p>在 CSS 中使用通配符：*</p>
<p>在 JavaScript 中判断类型：typeof x === 'string'</p>
```

### 3. 排版中的特殊符号

```html
<p>价格：&yen;199.00</p>
<p>copyright &copy; 2024 公司名称</p>
<p>3 &times; 5 = 15</p>
```

## 使用建议

1. **优先使用实体名称**：便于阅读和维护
2. **特殊字符用实体编号**：确保兼容性
3. **注意编码设置**：确保 HTML 文件使用 UTF-8 编码
4. **避免重复转义**：已转义的字符不要再转义
