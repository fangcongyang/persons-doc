# CSS 资源汇总

本文汇集了实用的 CSS 学习资源、工具库和灵感网站，帮助开发者提升 CSS 开发效率和创意。

## 样式技巧库

### CSS Tricks
[官网地址](https://qishaoxuan.github.io/css_tricks/)

- 总结一些常用的 CSS 样式
- 记录一些 CSS 的新属性和一点奇技淫巧
- 在"动"部分下有些动画并不是 CSS 效果，因为没有地方放置，所以暂时寄存在这里

### CSS-Tricks
[官网地址](https://css-tricks.com/)

- 最全面的 CSS 教程和技巧网站
- 涵盖从基础到高级的所有 CSS 主题
- 每日更新，包含最新的 CSS 特性和实践

## 生成工具

### 阴影生成器
- [CSS Box Shadow Generator](https://www.cssmatic.com/box-shadow) - 可视化生成 box-shadow
- [Neumorphism.io](https://neumorphism.io/) - 新拟态风格阴影生成器

### 渐变生成器
- [CSS Gradient](https://cssgradient.io/) - 功能强大的渐变生成器
- [Grabient](https://grabient.com/) - 精美的渐变收藏和生成工具

### 动画工具
- [Cubic-Bezier](https://cubic-bezier.com/) - 贝塞尔曲线调试工具
- [Animista](https://animista.net/) - 预置 CSS 动画库

### 布局工具
- [Grid Layout It](https://grid.layoutit.com/) - CSS Grid 可视化布局生成器
- [Flexbox Playground](https://flexbox.tech/) - Flexbox 交互学习工具

## CSS 框架

### 实用类框架
- **Tailwind CSS** - 原子化 CSS 框架，快速构建现代网页
- **Windi CSS** - Tailwind 的按需编译替代方案
- **UnoCSS** - 即时原子化 CSS 引擎

### 组件框架
- **Bootstrap** - 最流行的响应式前端框架
- **Bulma** - 基于 Flexbox 的现代 CSS 框架
- **Foundation** - 企业级前端框架

## 兼容性和测试

### 浏览器兼容性
- [Can I Use](https://caniuse.com/) - CSS 属性浏览器支持查询
- [Autoprefixer](https://autoprefixer.github.io/) - 自动添加 CSS 前缀

### 调试工具
- **Chrome DevTools** - 浏览器内置的强大 CSS 调试工具
- **Firefox DevTools** - 同样优秀的 CSS 网格和 Flexbox 检查器

## 性能优化

### 优化工具
- **PurgeCSS** - 移除未使用的 CSS
- **Critical** - 提取首屏关键 CSS
- **CSSNano** - CSS 压缩工具

### 最佳实践
- 使用 CSS 变量提高可维护性
- 优先使用 Flexbox 和 Grid 布局
- 合理使用 `will-change` 优化动画性能
- 避免过度使用 `*` 选择器和复杂嵌套
